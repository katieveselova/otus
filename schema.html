Данные о товарах
Name Definition Description
PK product_id bigint Уникальный идентификатор товара
name text Название товара, отображается на витрине и в поиске.
slug text ЧПУ-идентификатор для URL
brand text Бренд/производитель товара.
description text Текстовое описание товара

is_active boolean Признак опубликованности/доступности товара в каталоге
created_at timestamptz Время создания записи
updated_at timestamptz Время последнего обновления записи
