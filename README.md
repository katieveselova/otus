Реляционная модель данных для интернет‑магазина с товарами, вариантами, корзинами, заказами, платежами, складами и остатками. Репозиторий содержит схему и рекомендации по производительности и эксплуатации.

Что внутри
Чёткая предметная модель (товары → варианты → категории → корзина → заказ → платеж).
Гибкая расширяемость через JSONB‑атрибуты товара.
Продуманная стратегия индексов и ограничений целостности.
Рекомендации по транзакциям, резервированию остатков и TTL корзины.
